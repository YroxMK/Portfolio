Designed For NailSpa

Purpose: The Customer Feedback System was designed to collect feedback from customers using tablets set up on the client's premises. The system was specifically tailored to save all feedback details directly to a Google Sheet, using the Google Sheets API for seamless data integration

Main Features

  1. Branch-wise Employee Management
    • The setup was configured to save data in a branch-wise manner, categorizing feedback based on employees for each specific branch
    • The system dynamically displays the list of employees for the selected branch, allowing for targeted feedback collection
  2. Automated Response Handling
    • Automatically detects negative feedback
    • Sends an email notification with customer details and the review to the management for immediate attention
  3. Integration with Google Sheets
    • Utilized the Google Sheets API to ensure real-time data synchronization
    • Feedback data is structured to match the format of the Google Sheet, providing a smooth and organized data flow

Technology Stack

  • Backend: ASP.NET MVC, C#
  • Frontend: HTML, CSS, JavaScript
  • Integration: Google Sheets API
