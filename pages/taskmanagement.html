Client Mondo Migranti (Branches in Italy and Sri Lanka)

Year : Live since 2021 and still operational

Purpose: This system was developed to manage Mondo Migranti's entire operational process across its branches in Italy and Sri Lanka. It streamlines employee management, task workflows, customer registration, and financial updates

Main Features

  1. Employee Management
    • Attendance Tracking : Records employee attendance details across all branches
    • Leave Management : Manages employee leave requests and approvals efficiently
  2. Customer and Task Management
    • Customer Registration :Registers new customers into the system
    • Task Creation and Assignment :Allows the creation of tasks and assignment to specific employees or teams across branches
    • Task Updates :Tracks task progress and updates status in real-time
  3. Payment and Financial Management
    • Payment Updates :Records payment transactions related to different tasks
    • Comprehensive Process Coverage :The system handles all the financial aspects and ensures smooth task coordination between branches

Technology Stack

  • Backend: ASP.NET MVC, C#
  • Frontend: HTML, CSS, JavaScript
  • Database: SQL Server
