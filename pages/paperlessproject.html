Client Stafford Motors (Pvt) Ltd (Authorized Honda distributor in Sri Lanka)

Year : Live since 2021 and still operational

Purpose: PAS was developed to streamline the internal approval process of Stafford Motors. The system is designed to handle the approval workflows, reducing the reliance on paper-based methods.

Main Features

  1. User Levels
    • User : Creates requests and attaches relevant documents.
    • Manager : Reviews and approves requests before forwarding to top management
    • Top Manager : Provides the final approval.
    • Admin : Manages system configurations and user permissions.
  2. Request Types
    • The system supports 8 different types of requests, each with specific requirements
  3. Approval Workflow
    • Users create requests and attach relevant documents
    • Requests move to the Manager for review and approval
    • Approved requests are forwarded to the Top Manager for final approval
    • Once approved, users can download an Approval Sheet summarizing the key information
  4. Scalability
    • Currently used by over 200 users across all branches in Sri Lanka

Technology Stack

  • Backend: ASP.NET MVC, C#
  • Frontend: HTML, CSS, JavaScript
  • Database: SQL Server
