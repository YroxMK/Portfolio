Year : Live since 2024 and still operational

Purpose: The Internal Project Management System was developed to manage and streamline the company's internal project workflows. It facilitates efficient handling of client interactions, project assignments, milestones tracking, and financial processes

Main Features

  1. Client and Project Management
    • Client Registration : Ability to onboard new clients into the system
    • Project Creation : Create and manage projects, including assigning team members to each project
    • Milestones : Provides the final approval.
  2. Invoicing and Payment Management
    • Custom Invoice Generation :Generate invoices tailored to the project needs (initial, monthly, annually)
    • Automated Invoicing :Setup for automatic invoice generation based on project requirements
    • Payment Collection Updates :Track and update payment collections for each project.
  3. Financial Tracking
    • Manage payment statuses and keep a record of all transactions related to the projects

Technology Stack

  • Backend: ASP.NET MVC, C#
  • Frontend: HTML, CSS, JavaScript
  • Database: SQL Server
