Client Stafford Motors (Pvt) Ltd (Authorized Honda distributor in Sri Lanka)

Year : Live since 2023 and still operational

Purpose: The PMS was developed to facilitate the evaluation of employee performance within the company. It integrates with the PAS system and uses the same login credentials, operating as a separate module dedicated to performance assessments.

Main Features

  1. Integration with PAS:
    • PMS uses the same login credentials as PAS, allowing seamless access for users.
  2. Evaluation Process:
    • The HR department schedules performance evaluations for the employees
    • Employees can fill out their self-evaluation forms within the system
    • The Department Head reviews these self-evaluation forms and provides scores based on their assessments
    • The scores given by the Department Head are considered the final scores
  3. Role of HR
    • HR receives the finalized evaluations
    • HR downloads the necessary forms and initiates further processing based on the results

Technology Stack

  • Backend: ASP.NET MVC, C#
  • Frontend: HTML, CSS, JavaScript
  • Database: SQL Server
